# 💊 Drug Side Effect Predictor

A machine learning web app that predicts possible side effects 
of a drug based on patient parameters.

## 🎯 Features
- Predicts side effects based on Drug, Dosage and Age
- Random Forest Classifier (150 trees)
- Interactive web interface using Flask + HTML/CSS
- Instant prediction with confidence percentage

## 🛠️ Tech Stack
- Python
- Scikit-learn (Random Forest)
- Flask (Backend)
- HTML, CSS, JavaScript (Frontend)
- Pandas, NumPy

## 💊 Drugs Covered
Paracetamol, Ibuprofen, Aspirin, Cetirizine, Amoxicillin,
Diclofenac, Metformin, Azithromycin, Ciprofloxacin, Dolo650

## 🚀 How to Run
1. Install dependencies
   pip install flask scikit-learn pandas numpy
2. Run the app
   python app.py
3. Open browser
   http://127.0.0.1:5000
